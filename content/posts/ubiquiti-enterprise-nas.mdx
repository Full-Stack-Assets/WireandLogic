Ubiquiti has introduced a new Enterprise NAS (ENAS) built on the ZFS architecture, aiming to provide a private, local storage platform designed for performance, scalability, and simplicity. The ENAS combines powerful hardware with modern storage technologies to deliver enterprise-class performance. With 8 Arm Neoverse N2 cores, 64GB ECC memory, and dual NVMe cache, the ENAS is designed to handle demanding workloads. The system also features native ZFS for advanced storage protection and 16 drive bays for high-capacity deployments.

<Callout type="takeaway"> Ubiquiti's ENAS offers a scalable, cost-effective, and secure storage solution for enterprises.</Callout>

## What happened
The ENAS is built to grow alongside businesses, with extensive storage expansion capabilities and no firmware restrictions on drive models. The system integrates directly with the UniFi platform, creating a centralized source of truth for files, backups, and shared organizational data. The ENAS also features role-based access controls, integration with existing identity providers, and native secure access through UniFi Endpoint on desktop and mobile platforms.

The ENAS extends beyond traditional file storage with native iSCSI support in UniFi Drive, enabling shared block storage for virtualization environments and high-availability infrastructure. The system is designed to provide a simpler, easily scalable, and more cost-effective path to modern enterprise storage.

The launch of ENAS has been discussed on various forums, including Hacker News, where users have shared their experiences with ZFS and the need for a reliable and scalable storage solution. Some users have praised Ubiquiti's commitment to supporting their products, including a 15-year-old Edgerouter still receiving security updates.

## Why it matters
The introduction of ENAS matters because it provides a cost-effective and secure storage solution for enterprises. The system's scalability and performance make it an attractive option for businesses looking to upgrade their storage infrastructure. The use of ZFS architecture and native iSCSI support also make it an appealing choice for organizations with virtualization environments and high-availability infrastructure.

The ENAS also addresses the need for secure access controls and integration with existing identity providers. The system's role-based access controls and native secure access through UniFi Endpoint on desktop and mobile platforms provide an additional layer of security for organizations.
